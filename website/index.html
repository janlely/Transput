Transput输入法 GitHub

基于Rime引擎的Mac智能翻译输入法

Transput是一款基于Rime引擎开发的Mac智能翻译输入法，用户可选择模型并自定义翻译提示词实现输入之后的自动翻译。

即时翻译

输入完成后点击翻译按钮或使用Ctrl + Enter或/t(translate)调用翻译功能

翻译开启与关闭

输入开始前可以使用/s(switch)灵活控制翻译功能的开启与关闭

剪切板引入

输入/v命令自动引入剪切板内容

输入方案选择

输入/c(change schema)可切换输入方案

模型与Apikey设置

点击输入法菜单中的Settings打开设置窗口

中英混合输入

输入过程中使用Shift或/m(mode)切换中英模式
